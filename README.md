# 🚀 Login System with Blog Management

* 🏗️ Built using **Node.js, Express.js, MongoDB, Mongoose, EJS, Bootstrap 5, Passport.js, Express Session, and Cookie Parser**.
* 📂 Developed with a clean and scalable **MVC (Model-View-Controller) Architecture**.
* 🔐 Secure **User Authentication** with **Signup, Signin, and Logout** functionality.
* 🔑 Passwords are securely hashed using **bcrypt** before being stored in the database.
* 🍪 Implemented **Cookie Parser** to store, retrieve, and manage cookies.
* 🗂️ Configured **Express Session** for secure user session management.
* 🛡️ Integrated **Passport Local Strategy** with **Serialize** and **Deserialize** user functionality.
* 🚫 Protected private routes using custom **setAuthenticated Middleware**.
* 📝 Complete **Blog CRUD Operations** including Create, Read, Update, and Delete.
* 📋 Responsive Bootstrap table to manage blog records efficiently.
* 📄 Professional forms with server-side validation and user-friendly error messages.
* 🎨 Modern, responsive, and clean UI with reusable **Header** and **Footer** partials.
* 📊 Interactive Dashboard displaying blog statistics and user information.
* 📁 Proper organization of **Controllers, Models, Routes, Views, Middleware, Configurations, and Public Assets**.
* ⚡ Fully responsive design for desktop, tablet, and mobile devices.
* 🔒 Authentication-based access control ensures only authorized users can access protected pages.
* 🌐 Static assets are managed through Express Static Middleware.
* 🧹 Clean and modular codebase with **no comments** in the source files.
* ▶️ After installing dependencies and configuring the `.env` file,
*  start the application and open **http://localhost:5000** in your browser to access the project.
